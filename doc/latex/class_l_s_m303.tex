\subsection{\-Detailed \-Description}
\-Class to manage communication to the \hyperlink{class_l_s_m303}{\-L\-S\-M303} compass via the \-I2\-C-\/bus. 

\hyperlink{class_l_s_m303}{\-L\-S\-M303} has a 3-\/axis accelerometer and a 3-\/axis magnetometer on a single chip and the same \-I2\-C-\/bus. \-This class manages the interface to both of them and handles the read out procedure for the analog values. \-Check the data sheet for more details of the settings.
